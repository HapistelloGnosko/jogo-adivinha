# jogo-adivinha

Jogo de adivinhar número no terminal. Três níveis de dificuldade.

## Como usar

```bash
python jogo.py
```

## Níveis

| Nível  | Intervalo | Tentativas |
|--------|-----------|------------|
| Fácil  | 1 – 50    | 10         |
| Normal | 1 – 100   | 7          |
| Difícil| 1 – 200   | 5          |
