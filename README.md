# jogo-adivinha

`by Hapistello Gnosko`

</div>

---

Jogo de adivinhar número no terminal. Três níveis de dificuldade, feedback adaptativo e contagem de tentativas. Projeto focado em controle de fluxo e experiência do usuário no terminal.

---

## `// tech`

![Python](https://img.shields.io/badge/Python-0d0d0d?style=for-the-badge&logo=python&logoColor=6A0DAD)

---

## `// usage`

```bash
git clone https://github.com/HapistelloGnosko/jogo-adivinha
cd jogo-adivinha
python jogo.py
```

---

## `// levels`

| nível | intervalo | tentativas |
|---|---|---|
| Fácil | 1 – 50 | 10 |
| Normal | 1 – 100 | 7 |
| Difícil | 1 – 200 | 5 |

---

## `// features`

- Seleção de dificuldade antes de iniciar
- Feedback hot/cold a cada tentativa
- Contador de tentativas restantes
- Validação de entrada numérica
